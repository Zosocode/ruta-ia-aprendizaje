🧩 1. Introducción rápida: ¿Qué son las RNNs?

Las Redes Neuronales Recurrentes (RNNs) son una clase de redes diseñadas específicamente para procesar secuencias de datos, como texto, audio, series temporales, etc. A diferencia de redes feedforward, las RNNs tienen memoria interna, lo que les permite "recordar" información previa en una secuencia.

🛠️ 2. ¿Por qué TensorFlow + Keras para Deep Learning con RNNs?

🔧 TensorFlow (TF)

  • Framework de código abierto desarrollado por Google.

  • Muy poderoso, flexible y optimizado para entrenamiento en GPU/TPU.

  • Incluye herramientas como tf.data, tf.function, y TensorBoard.

🎛️ Keras

  • API de alto nivel integrada en TensorFlow (desde TF 2.0).

  • Permite construir modelos de manera más intuitiva, rápida y legible.

  • Ideal para prototipado y experimentación rápida.

⚠️ En resumen: Usamos Keras como interfaz, y TensorFlow como motor.

🧪 4. Código práctico: Construyendo una RNN con LSTM en Keras

python
import tensorflow as tf
from tensorflow.keras.models import Sequential
from tensorflow.keras.layers import LSTM, Dense, Embedding

# Parámetros
vocab_size = 10000
embedding_dim = 64
sequence_length = 100

# Modelo
model = Sequential([
    Embedding(vocab_size, embedding_dim, input_length=sequence_length),
    LSTM(128, return_sequences=False),  # También puedes usar return_sequences=True
    Dense(64, activation='relu'),
    Dense(1, activation='sigmoid')  # Para clasificación binaria
])

model.compile(optimizer='adam', loss='binary_crossentropy', metrics=['accuracy'])
model.summary()
